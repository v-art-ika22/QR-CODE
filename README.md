
# Terminal QR Code Generator 🚀

A sleek and efficient Node.js command-line tool that transforms any URL into a high-quality QR code image and saves a text log of the input. 

## 🛠 Features
* **Interactive CLI:** Uses `Inquirer.js` for a smooth, user-friendly command-line experience [cite: 2026-03-02].
* **Image Generation:** Leverages the `qr-image` library to create `.png` files instantly [cite: 2026-02-18].
* **Automated Logging:** Saves the source URL into a `.txt` file using Node's native `fs` module for future reference [cite: 2026-02-18].
* **Modern JavaScript:** Built using **ES Modules (ESM)** for better performance and alignment with modern Node.js standards [cite: 2026-02-18].

## 💻 Tech Stack
* **Language:** JavaScript (Node.js)
* **Environment:** Node.js v22.20.0+ [cite: 2026-02-18]
* **Libraries:** Inquirer, qr-image
